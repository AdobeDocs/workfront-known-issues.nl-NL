# Gebruikers: gebruikers die uit Adobe Admin Console zijn verwijderd, zijn nog steeds actief in Workfront

Wanneer een beheerder een gebruiker uit de Adobe Admin Console verwijdert, wordt die gebruiker niet uit Workfront verwijderd. De gebruiker is nog steeds zichtbaar en actief in Workfront.

_Eerste rapporteerde op 3 Februari, 2026._

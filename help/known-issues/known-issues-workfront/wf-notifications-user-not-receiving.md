# Meldingen: de gebruiker ontvangt geen verwachte meldingen

Wanneer een gebeurtenis voorkomt die een bericht naar een gebruiker zou moeten verzenden, ontvangt die gebruiker niet het bericht. Dit kan zelfs voorkomen als het plaatsen van het gebruikersbericht het bericht zou toestaan.

Dit probleem is van invloed op zowel in-app- als e-mailmeldingen.

_Eerste rapporteerde op 12 Februari, 2026._

# Gebruikers: kan gebruikerstoegang niet bijwerken

Wanneer een gebruiker probeert om de toegang van een andere gebruiker van Mening bij te werken om op een project te leiden, kan de gebruiker niet de toegang bijwerken, en zij zien het volgende bericht:

&quot;U hebt onvoldoende toegang om de financiering voor dit project te bekijken.&quot;

_Eerste rapporteerde op 11 December, 2025._

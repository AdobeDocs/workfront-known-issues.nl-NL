# Timesheets: Opmerkingen worden niet opgeslagen

>[!NOTE]
>
>Dit probleem is opgelost op 2 augustus 2022.

Wanneer een gebruiker een opmerking toevoegt aan een uuritem op een tijdspagina, wordt de opmerking niet opgeslagen.

_Voor het eerst gerapporteerd op 29 juli 2022._
